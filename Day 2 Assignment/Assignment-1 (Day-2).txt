7.How to use find(), findIndex() and filter() methods on Arrays in JavaScript? 


    find()
       
           -It returns the first element in an array that satisfies a provided testing function.

    findIndex()

           -It returns the index of the first element in an array that satisfies a provided testing function. If no element is found, it returns -1.

   filter()
 
          -It  creates a new array with all elements that pass the test implemented by the provided function.
